*** Closest Pair of Points, Running Instrutions ***

These scripts require the python 3 interpreter in order to function.
On the osu.flip servers, they can be run as follows: 

python3 bruteforce.py <inputfile>
python3 divideandconquer.py <inputfile>
python3 enhanceddnc.py <inputfile>

The outputs will be stored in files named as follows: 

output_bruteforce.txt
output_divideandconquer.txt
output_enhanceddnc.txt
